## Install the package

```bash [Terminal]
npm install nuxt-osdd
```

If you use pnpm, the equivalent command is:

```bash [Terminal]
pnpm add nuxt-osdd
```

## Update your `nuxt.config.ts`

Replace `defineNuxtConfig` with `defineOSDDNuxtConfig` and declare the layers your app depends on.

```ts [nuxt.config.ts]
import { defineOSDDNuxtConfig } from 'nuxt-osdd'

export default defineOSDDNuxtConfig({
  osdd: {
    technical: [...],
    functional: [...]
  },

  // your config
})
```

## Next step: create your first layer

Once the package is installed and your root `nuxt.config.ts` is updated, follow the dedicated [Creating Layers](/layers/creating-layers) guide to scaffold a `technical/` or `functional/` layer.

If you are migrating an existing app and want help deciding where code belongs, read the [migration guide](/getting-started/migration).
